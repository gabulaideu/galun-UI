title String 提示信息 弹出框标题（自定义弹出框无效）

btnSubmitText String 确定 确定按钮文字（自定义弹出框无效）
btnCancelText String 取消 取消按钮文字（自定义弹出框无效）

shadow Boolean true 是否显示遮罩层
opacity String 0.4 遮罩层透明度（0~1）

time Number 3000 自动关闭的时间（毫秒）
width Number 300 弹出框宽度（自定义弹出框无效）
effect String null

动画效果

zoom：缩放

fade：淡入淡出

slide：上下伸缩

scroll：顶部滑入
animation Number | String 500 动画效果时间（毫秒 / slow / normal / fast）
fixed Boolean true 是否固定定位
left Number null 设置离窗口左侧的距离（默认水平居中）

top Number null 设置离窗口顶部的距离（默认垂直居中）
bottom Number null 设置离窗口底部的距离（默认垂直居中）（设置底部距离时顶部距离自动忽略）
esc Boolean true 是否按esc键关闭弹出框
dblclick Boolean true 是否双击遮罩层关闭弹出框
lock Boolean false 是否锁定弹出框（即任何方式都无法关闭）
element String | Element null 自定义弹出框
submitCallback Function null 提交按钮的回调函数
cancelCallback Function null 取消按钮的回调函数
closeCallback Function null 关闭按钮的回调函数
shadowCloseCallback Function null 双击遮罩层关闭时的回调函数
escCallback Function null esc的回调函数
